# /board/meetings

## Propósito

Registro de cada reunión de nivel directivo de PITS MARKET. Incluye la agenda previa a la reunión y las minutas posteriores con decisiones y compromisos.

## Convención de Nombres

```
[YYYY-MM-DD]-meeting-[tipo]-[agenda|minutes].md

Tipos: weekly | monthly | investor | partner | emergency

Ejemplos:
  2026-07-25-meeting-weekly-minutes.md
  2026-08-01-meeting-investor-agenda.md
  2026-09-15-meeting-monthly-review-minutes.md
```

## Template de Minuta

```markdown
# Minuta de Reunión: [Tipo] — [Fecha]

**Duración:** X horas
**Participantes:** [Nombres y roles]
```

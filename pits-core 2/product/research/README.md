# /product/research — Investigación de Producto

## ⚠️ Diferencia con /research/

Esta carpeta contiene investigación **orientada a decisiones de producto** — lo que se aprende directamente para definir qué construir.

La carpeta `/research/` contiene investigación de mercado, competidores, y el Automotive Knowledge Graph.

| Esta carpeta `/product/research/` | La carpeta `/research/` |
|----------------------------------|------------------------|
| Sesiones de usabilidad de la plataforma | Análisis del mercado colombiano de autopartes |
| Tests A/B de features específicos | Benchmarks de competidores |
| Análisis de funnels de conversión | Entrevistas de usuario (archivo) |
| Feedback directo sobre el producto | Automotive Knowledge Graph |

## Qué va aquí

- Resultados de tests de usabilidad de flujos específicos (publicar, buscar, contactar)
- Análisis de conversión por pantalla con evidencia
- Hipótesis de producto validadas o refutadas con datos
- Reports de sesiones de user testing grabadas

## Convención de Nombres

```
usability-[flujo]-[YYYY-MM-DD].md
ab-test-[descripcion]-results-[YYYY-MM-DD].md
funnel-analysis-[etapa]-[YYYY-MM-DD].md
hypothesis-[numero]-[descripcion]-[VALIDATED|REJECTED].md

Ejemplos:
  usability-publish-flow-2026-08-15.md
  ab-test-cta-color-results-2026-09-01.md
  hypothesis-001-sellers-want-price-reference-VALIDATED.md
```

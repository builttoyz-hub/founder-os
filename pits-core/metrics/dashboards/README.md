# /metrics/dashboards — Dashboard Semanal

## Propósito

El dashboard operacional que el equipo revisa cada lunes. 15 métricas que explican el estado del negocio esta semana versus la semana anterior.

**Regla:** El dashboard se completa antes de las 10AM del lunes. Si no está completo, no empieza la semana.

## Convención de Nombres

```
[YYYY]-W[WW]-weekly-dashboard.md

Ejemplos:
  2026-W30-weekly-dashboard.md   ← Semana 30 de 2026 (julio 21–27)
  2026-W31-weekly-dashboard.md
  2026-W32-weekly-dashboard.md
```

# /brand/guidelines

## Propósito

Las reglas de uso de la marca PITS MARKET. Define qué se puede y qué no se puede hacer con el logo, los colores y la tipografía. Es la referencia para cualquier material de comunicación.

## Documento Principal

`brand-guidelines-v1.md` debe cubrir:

- Logo: versiones permitidas, espacio mínimo libre, tamaño mínimo
- Logo: usos prohibidos (distorsionar, cambiar colores, agregar sombras)
- Colores: uso primario vs secundario vs acento
- Tipografía: Barlow Condensed para UI, Share Tech Mono para datos técnicos
- Fotografía: estética oscura, carros reales, piezas reales, comunidad real
- Tono visual: NFS Underground pero premium, no barato

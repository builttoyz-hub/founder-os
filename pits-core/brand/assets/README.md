# /brand/assets

## Propósito

Los archivos exportados y listos para usar en producción. Nadie debería tener que abrir Figma para conseguir un logo. Todo lo que se necesita para usar la marca está aquí, ya exportado en los formatos y tamaños correctos.

## Archivos Obligatorios

```
logo-primary-1024.png      ← PNG con fondo transparente, 1024px
logo-primary-512.png       ← PNG con fondo transparente, 512px
logo-primary-256.png       ← PNG con fondo transparente, 256px
logo-transparent-512.png   ← Sin fondo, para uso sobre oscuro
logo-glow-512.png          ← Con efecto glow verde
logo-horizontal-1200.png   ← Versión horizontal para banners
favicon-32.png             ← Favicon 32x32
favicon-64.png             ← Favicon 64x64
og-default-1200x630.png    ← Open Graph por defecto
```

## Formatos

- **PNG** para uso en web con transparencia
- **SVG** para uso en código (escalable)
- **WebP** para uso optimizado en la plataforma

# /brand/identity

## Propósito

El núcleo del sistema de identidad visual: los logos en sus versiones originales (con capas editables), la paleta de colores con todos sus códigos, y las tipografías con sus especificaciones de uso.

## Archivos que van aquí

- Archivos de diseño editables (Figma, Illustrator) — los originales con capas
- Documentos que especifican el sistema: `colors-palette-v1.md`, `typography-system-v1.md`
- **NO** los exports — esos van en `/assets`

## Sistema de Color

| Nombre | Hex | Uso |
|--------|-----|-----|
| Verde Neón Principal | `#AAFF00` | CTAs, highlights, logo |
| Negro Profundo | `#0A0A0A` | Fondos principales |
| Verde Oscuro | `#1A7A1A` | Variante oscura para texto |
| Blanco Puro | `#FFFFFF` | Texto sobre fondos oscuros |
| Gris Medio | `#555555` | Texto secundario |

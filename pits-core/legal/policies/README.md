# /legal/policies

## Propósito

Las políticas internas que rigen la operación de PITS MARKET. No son contratos con usuarios, pero regulan cómo el equipo toma decisiones operacionales.

## Políticas Requeridas para el MVP

```
policy-content-moderation-v1.md       → ¿Qué se aprueba y qué se rechaza?
policy-prohibited-items-v1.md         → Lista de piezas que no se pueden vender
policy-dispute-resolution-v1.md       → Cómo se manejan conflictos entre usuarios
policy-account-suspension-v1.md       → Cuándo se suspende una cuenta y el proceso
policy-data-retention-v1.md           → Por cuánto tiempo guardamos qué datos
```

## Política de Ítems Prohibidos (Resumen)

Piezas que NO se pueden publicar en PITS MARKET:
- Piezas reportadas como robadas
- Piezas que modifiquen el odómetro o VIN
- Equipos de radar/jammers ilegales
- Piezas sin documentación de importación legal
- Réplicas vendidas como originales

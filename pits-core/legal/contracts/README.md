# /legal/contracts

## Propósito

Contratos firmados con proveedores de servicios, socios comerciales, influencers con acuerdos formales, y cualquier entidad con la que PITS MARKET tenga una relación contractual.

**⚠️ Todos los contratos deben revisarse por un abogado antes de firmar.**

## Convención de Nombres

```
contract-[contraparte]-[tipo]-[YYYY-MM].md

Tipos: services | influencer | partnership | employment | nda

Ejemplos:
  contract-revue-sas-influencer-2026-07.md
  contract-supabase-services-2026-06.md
  contract-designer-freelance-2026-08.md
```

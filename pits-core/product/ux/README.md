# /product/ux — Flujos y Experiencia de Usuario

## Propósito

Documenta **cómo se mueve el usuario** a través de PITS MARKET: los flujos paso a paso, wireframes y mapas de experiencia. Es el puente entre la especificación funcional y el diseño visual final.

## ¿Qué va aquí?

- **User flows:** Diagramas paso a paso de los flujos principales
- **Wireframes:** Bocetos de las pantallas (como imágenes PNG/SVG)
- **Customer journey maps:** El viaje completo del comprador y el vendedor
- **Decisiones de UX:** Por qué se eligió un patrón de UX sobre otro

## Convención de Nombres

```
[tipo]-[actor]-[accion]-v[N].[ext]

Tipos: flow | wireframe | journey | decision

Ejemplos:
  flow-seller-publish-listing-v1.md
  wireframe-marketplace-grid-v1.png
  journey-buyer-first-purchase-v1.md
  decision-single-page-form-vs-wizard.md
```

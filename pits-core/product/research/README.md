# /product/research — Investigación de Producto

## Propósito

La voz del usuario, documentada. Contiene entrevistas, encuestas, sesiones de usabilidad y cualquier evidencia directa de usuarios reales que informe las decisiones de producto.

> **Principio:** Las opiniones son gratuitas. La evidencia cuesta trabajo. La evidencia gana.

## Convención de Nombres

```
[tipo]-[descripcion]-[fecha].md

Tipos: interview | survey | usability | insight | persona

Ejemplos:
  interview-seller-medellin-bmw-2026-07-15.md
  survey-marketplace-ux-2026-08.md
  usability-publish-flow-2026-08.md
  insight-sellers-dont-verify-phone.md
  persona-comprador-tuning-colombia.md
```

## Plantilla de Entrevista

```markdown
# Entrevista de Usuario: [ID Anónimo o Alias]

**Fecha:** YYYY-MM-DD
**Duración:** X minutos
**Entrevistador:** [Nombre]
**Tipo de usuario:** Comprador / Vendedor / Ambos
**Ciudad:** [Ciudad]
**Vehículo(s):** [Si aplica]

## Contexto del Usuario
Descripción breve del perfil sin datos identificables.

## Preguntas y Respuestas
**P:** [Pregunta]
**R:** "[Respuesta textual o paráfrasis]"

## Insights Clave
1. [Insight #1]
2. [Insight #2]

## Citas Destacadas
> "[Cita textual relevante]"

## Implicaciones para el Producto
- [Acción o consideración derivada del insight]
```
